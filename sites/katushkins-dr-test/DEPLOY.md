# Как запустить квиз с сервером (синхронизация между телефонами)

Чтобы на всех устройствах отображалось, кто уже ответил, и правильные ответы открывались после того, как ответят все, нужен **сервер**. Без сервера квиз работает только на одном устройстве (localStorage).

---

## Вариант 1: Всё на одном хостинге с Node.js

Сервер раздаёт и страницы, и API. Все открывают **один и тот же адрес** — синхронизация работает без настроек.

1. Залей на хостинг папку проекта целиком (вместе с `server.js`, `package.json`, `quiz.html`, `index.html`).
2. На хостинге выполни: `npm install` и `npm start` (или укажи в настройках старт-команду `node server.js` и порт из переменной `PORT`).
3. Участники открывают в браузере адрес твоего приложения, например:  
   `https://твой-проект.railway.app/quiz.html`  
   Тогда запросы идут на тот же домен, `/api/state` и остальные API доступны, «кто ответил» и результаты совпадают на всех устройствах.

Подходящие хостинги: **Railway**, **Render**, **Fly.io**, **VPS** (с Node.js).

---

## Вариант 2: Фронт и сервер на разных сайтах

Страница квиза на одном домене (например GitHub Pages), API — на другом (например Railway).

### Шаг 1: Деплой сервера

- Залить на хостинг с Node.js только то, что нужно серверу: `server.js`, `package.json` (при желании можно не отдавать статику с этого сервера).
- Запустить: `npm install` и `npm start`.
- Запомнить URL приложения, например: `https://katin-quiz-api.railway.app`

### Шаг 2: Указать URL API в квизе

Перед подключением скрипта квиза задай базовый адрес API. В `quiz.html` в самом начале `<script>` (после открывающего тега) добавь одну строку:

```html
<script>
  window.QUIZ_API_BASE = "https://katin-quiz-api.railway.app";  // подставь свой URL сервера
</script>
<script>
  (function () {
    // ... остальной код квиза
```

Либо в консоли браузера один раз на странице квиза выполни (подставь свой URL):

```js
window.QUIZ_API_BASE = "https://твой-сервер.railway.app";
location.reload();
```

После этого квиз будет ходить за состоянием на твой сервер, и «кто ответил» будет синхронизироваться между всеми устройствами.

---

## Проверка

- Открой квиз на двух телефонах (или в двух вкладках) по **одному и тому же** адресу (вариант 1) или по адресу фронта с настроенным `QUIZ_API_BASE` (вариант 2).
- На одном устройстве введи имя и ответь на вопрос.
- На втором устройстве обнови страницу или подожди несколько секунд — должен появиться зелёный чип «кто уже ответил» и обновиться счётчик.

Если этого не происходит — в консоли браузера (F12 → Console) посмотри, нет ли ошибок при запросах к `/api/state` или к твоему URL API.
